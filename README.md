# Task List Application (Java)

A simple console-based Task List application written in Java.  
This program allows users to add, remove, and view tasks using a menu-driven interface.

---

## Features
- Add a new task
- Remove a task by task number
- Display all tasks
- Simple menu-driven console interface

---

## Technologies Used
- Java
- ArrayList (Java Collections Framework)
- Scanner for user input

---

## How It Works
1. The program displays a menu to the user.
2. The user selects an option from the menu.
3. Tasks are stored in an ArrayList in memory.
4. Task numbers start from 1 for easy selection.
5. The program continues running until the user chooses to quit.
